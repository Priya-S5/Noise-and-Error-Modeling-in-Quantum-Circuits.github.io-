ESP32 Microphone Streaming Project

Project Overview

This project demonstrates real-time audio streaming from a microphone to an ESP32 board and visualizing the audio data on a web interface. It is ideal for IoT audio monitoring, sound detection, and experimentation with streaming data.

Hardware Components

  • ESP32 Dev Board
  • Microphone Module (MAX9814 or KY-037)
  • Breadboard and Jumper Wires
  • Power Source (USB or Li-ion battery)
  • Optional: Capacitor 0.1ÂµF for noise filtering

Getting Started

Upload the ESP32 code using Arduino IDE or PlatformIO. Connect your ESP32 to Wi-Fi and access the streaming interface via a browser.

// Example Arduino snippet
#include <WiFi.h>
#include <ESPAsyncWebServer.h>

const char* ssid = "YOUR_SSID";
const char* password = "YOUR_PASSWORD";

void setup() {
  Serial.begin(115200);
  WiFi.begin(ssid, password);
  while (WiFi.status() != WL_CONNECTED) {
    delay(500);
    Serial.print(".");
  }
  Serial.println("Connected!");
}

void loop() {
  // Audio reading and streaming code
}
